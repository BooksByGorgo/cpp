# project description

create lulu executive book cover PDF for `Gorgo C++ and C` called `lulu-cover.pdf`.

## template

the template PDF is `lulu-cover-template.pdf`. descriptions of the parts and measurements are in the template.

## content

- use the c++c-gorgo-with-badge image for the front page and the title `lulu-cover-template.pdf` use the date of the last commit for the subtitle
- put the title on the spine with `Gorgo Book` and the base of the spine
- on the back put a cheat sheet of interesting tables from the sc++ and c4c++ books
